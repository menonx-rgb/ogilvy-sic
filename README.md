# Ogilvy Social, Influence and Content

A Vite + React website built for a strategic pitch deck presentation.

## Setup

1. Install Node.js and npm
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Deployment

Deploy on Vercel by importing this repository. Vercel should auto-detect the Vite setup.
